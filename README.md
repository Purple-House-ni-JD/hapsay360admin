## Getting Started

### Login / First-Time Access

Access to the **HAPSAY360 Admin Portal** requires administrator credentials created by an existing system administrator.

#### Login Process

1. Navigate to the **HAPSAY360 Admin Portal URL**
2. Enter your **Email Address**
3. Enter your **Password**
4. Click **Login** or press **Enter**

#### Forgot Password

1. Click **Forgot Password** on the login page
2. Enter your registered email address
3. Check your email for the reset link
4. Create a new password
5. Return to the login page and sign in

> **Note:** Admin accounts are created via **Stations & Personnel Management** by authorized administrators.
